## HelpyBot

> A Slack Bot

### Install

```
npm install
```

### Run

```
SLACK_TOKEN=[your slack token] npm start
```

### Scripts

- hi - Say hello!
- pug me - You won't regret it
- it's a trap - Display an Admiral Ackbar piece of wonder
